# 🔧 Quick Setup


## 📦 Required Libraries

### You must download and install the latest versions of the following libraries before using this script:

SD-Lib → https://github.com/Samuels-Development/sd_lib

ox_lib → https://github.com/overextended/ox_lib

#### ⚙️ Installation Steps

Download and install the latest versions of both SD-Lib and ox_lib.

Place the Black Market Van script inside your resources folder.

Make sure SD-Lib and ox_lib are started before this script in your server.cfg.

Edit your config.lua to customize:

Items

Prices

Locations

Currency mode (cash or item)

Start the resource and enjoy your new cinematic Black Market Van experience!
